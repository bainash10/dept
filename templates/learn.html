Project Overview

This project integrates deep learning models, including LeNet-5 and our custom-built model called Ran2Dev.

Prerequisites

  • Familiarity with Ranjana Lipi: Users must know the Ranjana Lipi script.
  • Image Condition: Characters should be written on plain paper with no strikes or additional markings.
  • Proper Alignment: Input images must be vertically aligned with minimal skew or no rotation.
  • No Modifiers: Only Ranjana characters without modifiers are supported.
  • Correct Input: The model gives incorrect results for non-Ranjana scripts.
  • Thick Strokes: The model is designed for characters with thick or medium strokes only.
